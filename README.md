# Shadcn/ui + Layouts

This component library is used to quickly scaffold application layouts and directives built on top of Shadcn/ui

## Technologies

1. React
2. Redux + shadcn/ui
3. Vite


## Run Changesets
[Learn more](https://github.com/changesets/changesets/tree/main#readme)

